// How to Sync Your Edits
Every time you make a change you want the world (or GitHub) to see, you have to follow this 3-step cycle:

1. Save the File: Press Ctrl + S. This saves the code to your computer's hard drive.

2. Commit: Go to the Source Control tab (where you saw the blue "Commit" button). Type a message like "Updated barracks cost" and click Commit. This "packages" your changes.

3. Sync/Push: Click the Publish Branch or Sync Changes button. This is the actual step that "uploads" the package to your GitHub repository.
